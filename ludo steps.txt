🎮 Ludo Game Basics
👥 Players
2 to 4 players
Each player has:
4 tokens/pawns
A home/base area
A colored path

Colors usually:

🔴 Red
🟢 Green
🔵 Blue
🟡 Yellow
🧩 Goal of the Game

Move all 4 tokens:

Out of the base
Around the board
Into the center/home triangle

First player to finish all 4 wins.

🎲 Dice Rules

Players roll:

1 dice (1–6)
🚪 Starting a Token

A token can leave the base ONLY when:

Player rolls a 6

Example:

Red player rolls 6
One red token moves from base to starting tile
🔁 Turn Flow
Step-by-step
1. Player Turn Starts

Example:

currentPlayer = "RED"
2. Roll Dice
dice = random(1,6)
3. Check Possible Moves

Player can:

Open new token if dice == 6
Move existing token
Skip if no move possible
4. Move Token

Example:

Dice = 4
Token moves 4 boxes forward
5. End Turn

Normally:

Next player turn

BUT…

If player rolls:

6 → gets another turn
🛣️ Board Path Logic

Each color has:

Different starting tile
Same circular path

Tokens move:

Around outer board
Enter own home path
Reach final center
🏠 Home Path

Near the end:

Token enters colored final lane

Example:

Red enters red lane

Then:

Must roll exact number to finish
💥 Killing / Capturing

If token lands on enemy:

Enemy token returns to base

Example:

Red lands on Blue
→ Blue goes back home
🛡️ Safe Zones

Some tiles are protected.

Usually:

Star tiles
Starting tiles

Enemy cannot kill there.

🎯 Winning

Player wins when:

all 4 tokens reached center
🧠 Important Game Rules
✅ Rule 1 — Roll 6
Opens token
Gives extra turn
✅ Rule 2 — Three 6s

Many versions:

3 continuous sixes = turn cancelled

Example:

6 → 6 → 6
= no move
next player turn

Optional rule.

✅ Rule 3 — Exact Finish

Need exact dice number to enter center.

Example:

2 tiles left
Dice = 3
→ cannot move
